Lo que he aprendido

HTML es un lenguaje de marcado que se utiliza para crear páginas web. Con HTML, puedo agregar texto, imágenes, videos y otros elementos a una página web. Algunas de las etiquetas HTML que he aprendido son:

html: Define el comienzo de un documento HTML.
head: Define la información de encabezado de un documento HTML.
title: Define el título de un documento HTML.
body: Define el cuerpo de un documento HTML.
h1: Define un encabezado de nivel 1.
p: Define un párrafo.
img: Define una imagen.
a: Define un hipervínculo.

También he aprendido a agregar atributos a las etiquetas HTML. Los atributos proporcionan información adicional sobre una etiqueta. Algunos ejemplos de atributos son:

src: Especifica la URL de la imagen.
href: Especifica la URL del hipervínculo.
alt: Proporciona un texto alternativo para la imagen.

Además, he aprendido a crear listas, tablas y formularios en HTML. Las listas se pueden crear utilizando las etiquetas ul y li para listas sin ordenar y las etiquetas ol y li para listas ordenadas. Las tablas se pueden crear utilizando las etiquetas table, tr, th y td. Los formularios se pueden crear utilizando la etiqueta form y otros elementos como input y button.

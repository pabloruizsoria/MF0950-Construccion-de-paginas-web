Booleanos

Los booleanos pueden tomar 2 posibles valores: true (verdadero) o false (falso)

El operador AND se escribe con &&

El operador OR se escribe con ||

En una operación AND se obtiene true si todas las entradas son true y false en otro caso.

En una operación OR se obtiene true si cualquiera de las entredas es true y false en cualquier otro caso

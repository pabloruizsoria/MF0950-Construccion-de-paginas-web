Ejercicio 1

Pide al usuario 2 números

Pide al usuario qué hacer con dichos números. Por ejemplo: Si te proporciona un + suma ambos. Si te proporciona un - réstalos. Si introduce un * entonces multiplícalos. Si escribe / realiza la división.

Debes usar funciones para resilver el ejercicio. Cada función escribirá el resultado en consola.

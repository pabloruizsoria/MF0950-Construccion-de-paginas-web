Ejercicio 2: Descuentos

Pide que introduzcan el precio de un producto (número mayor que 0)

Pide que introduzcan un descuento (número entre 0 y 100)

Comprueba que hayan introducido datos correctos

Muestra en consola

  • El precio del producto
  • El descuento a aplicar
  • En cuanto se queda el producto tras el descuento

Ejemplo

  • El precio del producto: 1.000 €
  • Descuento: 20%
  • El descuento a aplicar: 200€
  • En cuanto se queda el producto tras el descuento: 800€
